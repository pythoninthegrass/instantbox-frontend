# instantbox frontend

## Introduction

This repository hosts the frontend component of [instantbox](https://github.com/pythoninthegrass/instantbox).

## Setup

### Clone

```bash
git clone https://github.com/instantbox/instantbox-frontend.git
```

### Develop

```bash
npm install
npm start
```

### Test

```bash
npm run lint
npm test
```

### Package

```bash
npm run build
```
